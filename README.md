2. Short Description
   
This project presents an interactive E-Commerce Sales Dashboard created using Power BI.
It visualizes key business metrics such as total sales, profit, customer performance, category-wise quantity, state-wise revenue, payment mode distribution, and monthly profit trends.
The dashboard helps identify business insights and patterns that support data-driven decision-making.
